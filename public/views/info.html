Welcome!

This website was developed by a group of students at Missouri State University for their CSC 450 - Software Engineering class project.

The main goal of this website is to find out if certain user's gravitate towards a certain image set when they choose what images they want.

No personal data is required but it is strongly encourage to provide at least a gender and an age. This will help us determine if images can be used as passwords for banking, personal applications, and much more!

Instructions for the game:

1. Select an image set that is 5 images long

2. Press begin and quickly select your image set from the images shown

3. If you select the correct images in the alloted time, you will advance to the next level with increased difficulty

Good luck!
